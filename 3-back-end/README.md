# BACK-END

Nesse módulo aprendi sobre desenvolvimento de aplicações no lado do servidor, utilizando `Node.js` e o framework `Express`. Durante os estudos, entendi como criar APIs REST, organizar rotas, controlar requisições HTTP e estruturar aplicações back-end de forma escalável.
Na sequência aprendi sobre `bancos de dados relacionais`, principalmente utilizando `MySQL`, além de escrever consultas com `SQL` para manipular e recuperar dados. Também aprendi sobre modelagem de banco de dados, criação de tabelas, relacionamentos e normalização.
Além disso, estudei o uso de `ORMs`, autenticação e autorização de usuários, além da criação de testes automatizados para garantir a qualidade e o funcionamento das aplicações.
Com isso, me tornei capaz de desenvolver APIs completas, realizar integração entre aplicações e bancos de dados, aplicar boas práticas de arquitetura e garantir segurança básica nas aplicações.
Também aprendi a organizar projetos seguindo padrões profissionais utilizados no mercado de tecnologia.
Tendo isso em mente, esta pasta contém todas as atividades de aprendizagem que desenvolvi ao longo do módulo de `Back-end` do curso da [Trybe](https://www.betrybe.com/).
